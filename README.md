
# 🦹🏻Books Mafia 
Books Mafia is a website where you can add, delete or update any book and list it for sale at desired price.
## ⚡Tech Stack

**Client:** React, CSS

**Server:** NodeJS, Express

**Database:** MySQL

  
## 💻Features

- Add a book
- Delete a book
- Update a book
